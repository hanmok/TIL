# Join


## INNER JOIN 
- SELECT * from A JOIN B ON condifition
- Select all records from A and B where the join condition is met
- DEFAULT Join

## LEFT JOIN 
- SELECT * from A LEFT JOIN B ON condifition
- Select everything from A, along with any matching records in B
